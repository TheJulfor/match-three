# Три в ряд

Это консольная версия игры "Три в ряд" на языке C++.
Целью является набрать как можно больше очков, меняя местами соседние элементы для создания комбинаций из трех и более одинаковых символов.

## Особенности

- Комбо-ходы: при удалении выигрышных элементов, есть вероятность что сгенерируются новые комбинации, которые также учитываются в статистику.
- Отмена хода: если при перемещении элементов никакая комбинация не сложилась, ход отменяется.
- Хорошая генерация в начале: перед началом игры доска очищается так, чтобы игра не началась с уже готовыми комбинациями на доске.

## Как играть

После запуска игры будет показана доска. Нужно будет ввести координаты тех элементов которые хочется поменять.
Эти элементы сменятся местами и будут высчитаны очки в статистику.
Для выхода из игры, введите 0 во время ввода координат первого элемента.

## Структура игры

- Main.cpp - основной файл запуска, меню
- Matrix.cpp - работа с игровой доской
- Management.cpp - логика игры
